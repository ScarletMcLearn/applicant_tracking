Applicant Tracking System 
=========

##  Recruit the best talent. Build great teams.




***Description:***
Applicant Tracking System helps ease the workload of Recruiters and Hiring managers. It is a simple way to reach more qualified candidates and track, share, and evaluate applicant information. It is not only a filtering database, but a great organizational tool for hiring. Hiring managers can send emails directly to applicants to either inform them that they didnâ€™t make the cut, or ask for an interview.

**How to set up the code**

1. Install [Node.js](https://nodejs.org/en/download/) and [MongoDB](https://www.mongodb.com/download-center?jmp=nav) on your computer

2. Download this code, and from command prompt run:

   `npm install`


   `bower install`


3. To run the code, run:

    `node server.js`
